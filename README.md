# Auto Mail Campaign Manager

![Logo](public/logo.svg)

Auto Mail Campaign Manager is a powerful and efficient tool designed to automate your email outreach campaigns. Leveraging AI, it allows you to send personalized emails automatically from your Google account without hitting spam filters. With features like CSV file upload for recipient lists, AI-powered mail template generation, and automated sending loops, you can manage large-scale campaigns with minimal oversight.

## Key Features

- **Automated Email Sending**: Set up email campaigns that run automatically, saving you time and effort.
- **AI-Powered Mail Templates**: Generate or select professional and engaging email templates with the help of artificial intelligence.
- **Credential Management**: Securely log in and manage your Google account credentials for seamless email sending.
- **CSV Integration**: Easily upload recipient lists via CSV files.
- **Spam Filter Avoidance**: Designed to send emails effectively, reducing the likelihood of them being marked as spam.
- **High Volume Sending**: Capable of sending approximately 1000 emails in 3.5 days on an automated loop.
- **Set-and-Forget Campaigns**: Once a loop is started, Auto Mail Campaign Manager handles the sending process autonomously.

## Technology Stack

This project is a [Next.js](https://nextjs.org) application bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app). It utilizes modern web technologies to deliver a robust and scalable solution.

## Getting Started

Follow these steps to get your development environment up and running:

### Prerequisites

- Node.js (v18 or later recommended)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/automail-campaign-manager.git
   cd automail-campaign-manager
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

### Running the Development Server

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
